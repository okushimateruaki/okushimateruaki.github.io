目的

本講義の目的は、
  1. 記述統計(1章)
  2. 確率分布(2章)
  3. 推定(3章)
  4. 検定(4章)
について学ぶことにあります。ここで、( )の中の章番号は教科書の章番号です。 意味を理解した上で実際に具体的な計算ができるようになることを目指します。

概要:

本講義では板書による説明の後、適宜例題演習を行います。

教科書

教科書は、 小寺平治著「ゼロから学ぶ統計解析 (ゼロから学ぶシリーズ)」 です。 授業時に各自持参すること。

内容

  1. 記述統計
    1. 度数分布表
    2. ヒストグラム
    3. 平均
    4. 分散
    5. 相関係数
  2. 確率分布
    1. 確率変数、確率分布
    2. 期待値　\(E(X)\)
    3. 同時確率変数
    4. 独立性
    5. 二項分布　\(B(n,p)\)
    6. 正規分布　\(N(\mu,\sigma^2)\)
  3. 推測統計
    1. 母集団、標本調査
    2. 標本平均の分布
    3. 標本分散の分布
    4. 母平均の推定
    5. 母分散の推定
    6. 母比率の推定
  4. 仮説検定
    1. 母平均の検定
    2. 母分散の検定
    3. 母比率の検定
    4. 適合度の検定（メンデルの法則の検定、独立性の検定）
    5. 母相関係数の検定
