目的

本講義の目的は、微分積分学や線形代数を学ぶときに必要となる知識を確実に身につけることにあります。

概要:

本講義では板書による説明の後、適宜例題演習を行います。 教科書は、 奥島, 小林, 竹中, 奥村「基礎数学」培風館 です。 授業時に必ず各自持参すること。

内容

  1. 数と数式
    1. 実数：有理数、分数の計算、無理数
    2. 複素数
    3. 整式：式の加法乗法、展開、因数分解
    4. 整式の除法と分数式
  2. 方程式と不等式
    1. 一次方程式、二次方程式
    2. その他の方程式
    3. 一次不等式
    4. 二次不等式
    5. 連立方程式
  3. 関数
    1. 有理関数：一次関数、二次関数; 高次関数; 分数関数; 関数のグラフと不等式
    2. 無理関数
    3. 絶対値関数
    4. 逆関数
    5. 指数関数
    6. 対数関数
    7. 三角関数
    8. 偶関数と奇関数：\(y=f(x)\)が\(f(-x)=f(x)\)なら偶関数； \(f(-x)=-f(x)\)なら奇関数
    9. 合成関数
    10. 関数の媒介変数表示

  4. 平面図形と方程式
    1. 点と直線
    2. 二次曲線
    3. 領域とグラフの移動

  5. 整式の微分と積分
    1. 微分法
    2. 整式の微分
    3. 微分法とグラフ描画
    4. 積分法
    5. 定積分と面積

70点以上79点以下をB
80点以上89点以下をA
90点以上100点以下をS
と成績評価します。
この点数が59点以下の場合には、 7/26日の再評価試験を受験してもらいます。
再評価試験の結果が60点以上の場合には Cの成績評価となり、
59点以下の場合には単位はありません。
